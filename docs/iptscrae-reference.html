PalaceV Iptscrae Reference

Complete reference for the Iptscrae scripting language as implemented in PalaceV. Iptscrae is a stack-based (Forth-like / PostScript-like) language using reverse-polish notation (RPN). All values are pushed onto the stack and commands pop their arguments from it. Comments begin with ; or #.

Syntax & Stack Basics

How the Stack Works

Values are pushed left-to-right, then commands consume them. Iptscrae uses reverse-polish notation.

; Push 10 and 20, add them, convert to string, display
10 20 + SAY   ; says "30"

; String concatenation
"Hello " USERNAME & SAY   ; says "Hello Jamie"

; Conditionals — the atomlist (code block) comes BEFORE the condition
{ "You're a harry wizard!" SAY } ISWIZARD IF

; Atomlists (code blocks) are delimited by { }
{ "true branch" SAY } { "false branch" SAY } x 5 == IFELSE

; While loop — body atomlist, then condition atomlist
{ i LOGMSG  i++ } { 10 i > } WHILE

; Variable assignment with =
42 myVar =
"hello" greeting =

Events

Events are declared with ON eventname { ... } in spot or cyborg scripts. Special variables are automatically available inside the event handler.

Room Lifecycle

ON ENTER event

Fires when you enter a room. This is the classic room entry event.

ON ENTER {
    "Welcome to " ROOMNAME & "!" & SAY
}

ON LEAVE event

Fires when you leave the current room.

ON LEAVE {
    ; Clean up alarms before leaving
    STOPALARMS
    HTTPCANCEL
}

ON SIGNON event

Fires when you first connect to a server.

ON SIGNON {
    "Connected to " SERVERNAME & LOGMSG
}

ON SIGNOFF event

Fires when you disconnect from the server.

ON SIGNOFF {
    "Disconnected." LOGMSG
}

ON ROOMLOAD event extended

Fires right before the room is rendered. Useful for making local changes to the display seamlessly.

ON ROOMLOAD {
    ; Modify spot appearance before anything is drawn
    1 AUTOUSERLAYER
}

ON ROOMREADY event extended

Fires once the room's background image has been downloaded and processed. ROOMWIDTH and ROOMHEIGHT are only accurate inside this event.

ON ROOMREADY {
    "Room size: " ROOMWIDTH & "x" & ROOMHEIGHT & LOGMSG
}

HTTP Events

ON HTTPRECEIVED event extended

Fires in the spot where HTTPGET or HTTPPOST was called when content is received. Some content types (iptscrae, images, html) are processed automatically after. Set CONTENTS to "" to prevent automatic processing.

VariableDescription
CONTENTSThe received content (atomlist if text/iptscrae, string otherwise). Read/write.
HEADERSThe HTTP response headers as a Hash element
TYPEThe Content-Type of the response
FILENAMEThe filename of the received file
ON HTTPRECEIVED {
    ; Check the content type and log it
    "Received: " TYPE & " (" & FILENAME & ")" & LOGMSG
    {
        ; Save image with custom name
        "custom.png" CONTENTS IMAGETOMEDIA
        "" CONTENTS =   ; Prevent double-save
    } TYPE "image/png" REGEXP IF
}

ON HTTPERROR event extended

Fires when a download error occurs after HTTPGET or HTTPPOST.

VariableDescription

ON HTTPERROR {
    "HTTP Error: " ERRORMSG & LOGMSG
}

ON HTTPRECEIVEPROGRESS event extended

Fires when new data is received from an HTTP connection.

VariableDescription

TOTALBYTESTotal expected bytes
ON HTTPRECEIVEPROGRESS {
    "Downloaded " BYTESRECEIVED & "/" & TOTALBYTES & LOGMSG
}

ON HTTPSENDPROGRESS event extended

Fires when data is being sent from an HTTPPOST connection.

VariableDescription

ON HTTPSENDPROGRESS {
    "Sent " BYTESSENT & " bytes" & LOGMSG
}

Commands

Control Flow

IF classic

Executes an atomlist if the condition is true (non-zero).

atomlist
Code block to execute if true
condition
Integer — non-zero is true
# IF — conditional execution
{ "You're a wizard!" SAY } ISWIZARD IF

IFELSE classic

Executes one of two atomlists based on a condition.

trueAtomlist
Code block for true
falseAtomlist
Code block for false
condition
Integer — non-zero is true
# IFELSE — if/else branching
{ "Wizard!" SAY } { "Not a wizard." SAY } ISWIZARD IFELSE

WHILE classic

Loops executing the body atomlist as long as the condition atomlist evaluates to true.

bodyAtomlist
Code block to execute each iteration
conditionAtomlist
Code block that pushes a boolean
# WHILE — loop from 0 to 9
0 i =
{ i LOGMSG  i++ } { 10 i > } WHILE

FOREACH classic

Iterates over each element of an array, pushing each element onto the stack before executing the body.

bodyAtomlist
Code to run per element (element is on the stack)
array
The array to iterate over
# FOREACH — iterate an array
3 ARRAY myArr =
"a" myArr 0 PUT  "b" myArr 1 PUT  "c" myArr 2 PUT
{ LOGMSG } myArr FOREACH

EXEC classic

Executes an atomlist from the stack.

atomlist
The code block to execute
# EXEC — execute a code block
{ "Hello!" SAY } myCode =
myCode EXEC

BREAK classic

Breaks out of the current WHILE or FOREACH loop.

# BREAK — exit loop early
{ { BREAK } i 5 == IF  i++ } { 10 i > } WHILE

RETURN classic

Returns from the current atomlist execution immediately.

# RETURN — exit current code block
{ { RETURN } x 0 == IF  "Continuing..." LOGMSG } EXEC

EXIT classic

Halts all script execution for the current event.

# EXIT — stop all script execution
{ EXIT } dangerousCondition IF

Stack Operations

DUP classic

Duplicates the top item on the stack.

# DUP — duplicate top of stack
42 DUP  ; stack: 42 42

POP classic

Removes the top item from the stack.

# POP — discard top of stack
1 2 3 POP  ; stack: 1 2

SWAP classic

Swaps the top two items on the stack.

# SWAP — swap top two items
1 2 SWAP  ; stack: 2 1

OVER classic

Copies the second item to the top of the stack.

# OVER — copy second item to top
1 2 OVER  ; stack: 1 2 1

PICK classic

Copies the Nth item from the stack to the top.

n
Zero-based index from top of stack
# PICK — copy Nth item
10 20 30 2 PICK  ; stack: 10 20 30 10

STACKDEPTH classic

Pushes the current number of items on the stack.

# STACKDEPTH — check stack size
1 2 3 STACKDEPTH LOGMSG  ; logs "3"

Variables

= (Assign) operator

Assigns the top stack value to a variable. Also available as DEF.

# = — variable assignment
42 myVar =
"Hello" greeting =

GLOBAL classic

Declares a variable as global, persisting across events within the same scope (room or cyborg). Alias: SGLOBAL.

variableName
The name of the variable (used after it, e.g. score GLOBAL)
# GLOBAL — persistent variable
score GLOBAL
score 1 + score =

TOPTYPE classic

Pushes the type of the top stack item as an integer without consuming it.

# TOPTYPE — check what's on the stack
"hello" TOPTYPE LOGMSG

VARTYPE classic

Pushes the type of a variable as an integer.

# VARTYPE — check variable type
myVar VARTYPE LOGMSG

String Operations

ATOI classic deprecated

Converts a string to an integer. Deprecated — type conversion is automatic.

# ATOI — string to integer
"42" ATOI  ; pushes 42

; No longer needed — strings auto-convert to numbers
"10" 5 +  ; result is 15

ITOA classic deprecated

Converts an integer to a string. Deprecated — type conversion is automatic.

# ITOA — integer to string
42 ITOA  ; pushes "42"

; No longer needed — numbers auto-convert to strings
"Score: " 100 & SAY  ; says "Score: 100"

STRLEN classic

Pushes the length of a string.

# STRLEN — string length
"Hello" STRLEN SAY  ; says "5"

STRINDEX classic

Finds the index of a substring within a string. Returns -1 if not found.

# STRINDEX — find substring position
"lo" "Hello" STRINDEX SAY  ; says "3"

SUBSTR classic

Extracts a substring by start position and length.

string
Source string
start
Start index
length
Number of characters
# SUBSTR — extract substring
"Hello World" 6 5 SUBSTR SAY  ; says "World"

SUBSTRING classic

Extracts a substring by start position and length (alternate form).

# SUBSTRING — extract substring
"Hello" 0 3 SUBSTRING SAY  ; says "Hel"

LOWERCASE classic

Converts a string to lowercase.

# LOWERCASE — to lower
"HELLO" LOWERCASE SAY  ; says "hello"

UPPERCASE classic

Converts a string to uppercase.

# UPPERCASE — to upper
"hello" UPPERCASE SAY  ; says "HELLO"

STRTOATOM classic

Parses a string as Iptscrae code and pushes the result as an atomlist.

# STRTOATOM — parse string as code
"\"Hello!\" SAY" STRTOATOM EXEC

CHARTONUM extended

Returns the ASCII code of a character.

# CHARTONUM — char to ASCII code
"A" CHARTONUM LOGMSG  ; logs "65"

NUMTOCHAR extended

Returns the character for an ASCII code.

# NUMTOCHAR — ASCII code to char
65 NUMTOCHAR SAY  ; says "A"

REPLACE extended

Replaces the first occurrence of a string.

oldString
String to find
newString
Replacement string
sourceString
Original string
# REPLACE — replace first occurrence
"it" "this" "I find this handy." REPLACE SAY

REPLACEALL extended

Replaces all occurrences of a string.

# REPLACEALL — replace all occurrences
" " "_" "Hello World Test" REPLACEALL SAY  ; "Hello_World_Test"

GREPSTR classic

Tests a string against a regex pattern. Pushes 1 if matched, 0 otherwise. Patterns are filtered for backward compatibility.

# GREPSTR — regex match test
{ "Contains hello!" SAY } "hello" CHATSTR GREPSTR IF

GREPSUB classic

Regex substitution with up to 10 matches.

# GREPSUB — regex replace
"world" "hello" "hello world" GREPSUB SAY

REGEXP extended

Like GREPSTR but uses full PCRE regex without backward-compatibility filtering.

# REGEXP — full regex match
{ "Match!" SAY } "^[A-Z]+" "HELLO" REGEXP IF

REGEXPREPLACE extended

Like GREPSUB but not limited to 10 matches.

# REGEXPREPLACE — unlimited regex replace
"_" "\\s+" "Hello   World" REGEXPREPLACE SAY

ENCODEURL extended

URL-encodes a string for safe use in HTTP requests.

# ENCODEURL — URL encode
USERNAME ENCODEURL SAY

DECODEURL extended

Decodes a URL-encoded string.

# DECODEURL — URL decode
"Hello%20World" DECODEURL SAY  ; says "Hello World"

Array & Hash

ARRAY classic

Creates a new array of the specified size.

size
Number of elements
# ARRAY — create an array
5 ARRAY myArr =
"first" myArr 0 PUT

GET classic

Gets a value from an array by index, or from a hash by key.

container
Array or hash
indexOrKey
Integer index or string key
# GET — read from array or hash
myArr 0 GET SAY
myHash "name" GET SAY

PUT classic

Stores a value into an array at an index, or into a hash by key.

value
The value to store
container
Array or hash
indexOrKey
Integer index or string key
# PUT — write to array or hash
"hello" myArr 0 PUT
"Jamie" myHash "name" PUT

LENGTH classic

Pushes the length of an array.

# LENGTH — array length
myArr LENGTH SAY

NEWHASH extended

Creates a new empty hash (dictionary/object).

# NEWHASH — create a hash
NEWHASH myhash =
"bar" myhash "foo" PUT
myhash HASHTOJSON LOGMSG  ; {"foo":"bar"}

HASHTOJSON extended

Converts a hash to a JSON string.

# HASHTOJSON — hash to JSON
myhash HASHTOJSON LOGMSG

JSONTOHASH extended

Parses a JSON string into a hash element.

# JSONTOHASH — JSON to hash
"{\"name\":\"Jamie\"}" JSONTOHASH myhash =
myhash "name" GET SAY  ; says "Jamie"

Arithmetic & Assignment Operators operator

Standard arithmetic, comparison, logical, and assignment operators.

# Arithmetic
10 5 +     ; 15 (addition)
10 5 -     ; 5 (subtraction)
10 5 *     ; 50 (multiplication)
10 5 /     ; 2 (division)
10 3 %     ; 1 (modulo)

# String concatenation
"Hello " "World" &   ; "Hello World"

# Comparison
5 3 ==     ; 0 (equal)
5 3 !=     ; 1 (not equal, also <>)
5 3 >      ; 1 (greater than)
5 3 <      ; 0 (less than)
5 3 >=     ; 1
5 3 <=     ; 0

# Logical
1 1 AND    ; 1
1 0 OR     ; 1
1 NOT      ; 0 (also !)

# Shortcuts
i++        ; increment variable
i--        ; decrement variable
5 i +=     ; add and assign
3 i -=     ; subtract and assign
2 i *=     ; multiply and assign
2 i /=     ; divide and assign
3 i %=     ; modulo and assign
"!" s &=   ; concat and assign
